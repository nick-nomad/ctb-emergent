## 🔧 How to Use This Template for Other Products

### Step 1: Copy the File
```bash
Copy garden-rooms.html → garages.html
Copy garden-rooms.html → stables.html
Copy garden-rooms.html → sheds.html
(etc.)
```

### Step 2: Update These Sections

**In the `<head>`:**
```html
<title>Change the title</title>
<meta name="description" content="Change description">
```

**Hero Section:**
- Change the h1 heading
- Update subtitle/phone call text
- Change hero image URL

**Gallery Images:**
- Replace 4 image URLs with relevant product images

**Main Content:**
- Update introduction paragraphs
- Modify feature lists
- Change section headings
- Update FAQ questions/answers

**Keep the Same:**
- Header navigation
- Footer
- Overall layout structure
- Styling and formatting
